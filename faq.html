What do you mean by WordPress integration? Do you create a WordPress site or a combo of WordPress and HTML/CSS?

Usually we deliver just the WordPress theme code unless our client requests the static HTML files explicitly. Upon request we can provide an archive with the static files used for creating the WordPress theme. Please note that any changes requested to the WordPress site won't be applied to the static version of the files.

Can you help with the installation?

Sure. We need the following details in order to install the theme on your server:

  1. FTP or SFTP credentials (host, username, password, target directory)
  2. URL address of the new installation

We can also install WordPress using other methods (e.g. remote desktop, VPN).

Please note that we don't require hosting control panel or phpMyAdmin access for installation purposes.

What exactly does "WordPress Integration" involve? Do you return code as a ready to install theme?

We deliver an archive of the complete WordPress site. The archive includes:

  4. Any files uploaded during the initial theme development and associated with the example content

How can I send my email template?

You can send your template using a desktop email client such as Outlook, Thunderbird or you can use a web based email marketing tool like MailChimp or Campaign Monitor.

How can I import my email template to MailChimp?

It's easy just follow the steps below.

  1. Login to MailChimp.
  2. Once you have logged in, click on "my templates" next to the "create campaign" big red button.
  3. Click on "code custom template" in the sidebar, below "Code".
  4. Click on "import zip file" at the top.
  5. Click on "browse", select mc.zip from the archive we have sent you and click on "upload".
  6. You're almost there. Now click on "save", type the name of the template and click on "save as".
  7. You're done! Now you can find your template under "My Templates".

If you have any problems, just contact us. We'll import it free of charge.

How can I import my email template to Campaign Monitor?

Below are the steps you need to take in order to import your template to Campaign Monitor:

  1. Login to Campaign Monitor.
  2. Select a "Client you're sending" on the left.
  3. Click on "Templates" at the top right corner.
  4. Click on "Add a new template" on the right sidebar.
  5. On the popup window click on "Import a custom template" - the "Import it now" button.
  6. "Template Name" - type the name of your template.
  7. "The HTML page" - click on the "Browse" button, go to the source files we have sent you - folder "cm" and select email.html.
  8. "All other files in zip format (optional)" - click on the "Browse" button, go to the source files we sent you, "cm" folder and now select images.zip.
  9. Click on "Add template".
  10. You're done! Go get some burgers.

If you have any problems, just contact us. We'll import it free of charge.

How can I send my email template via Thunderbird?

Here is how:

  1. First you have to upload all the images to your hosting. Login to your server via FTP, go to the root folder usually "public_html", "www" or "html". Create folder for your email images say "mail-images".
  2. Upload all the images from the archive we have sent you, folder "images" to your FTP folder "mail-images".
  3. Now open email.html with any text editor (notepad for example) from the archive we have sent you.
  4. Replace all image source paths, so all images point to your hosting - http://yourdomain.com/mail-images/image.gif. You can do that by finding and replacing - find "images/" and replace it with "http://yourdomain.com/mail-images/".
  5. Now that you have changed the images, open email.html with your browser to make sure all the images are loaded.
  6. Open Thunderbird and click on "Write".
  7. Click on the message area and then click on "Insert" - "HTML". Copy all the html code from email.html, paste it here and click on "Insert".

It's that simple! Now you're ready to send your html email templates via Thunderbird.

It's pretty much the same as Thunderbird.

  2. Upload all the images from the archive we have sent you, folder "images" to your FTP folder "mail-images".
  3. Now open email.html with any text editor (notepad for example) from the archive we have sent you.

You're now ready to send your template!
